Generate a github fine-grained secure token in less than 1 minute



1. Navigate to your Git account settings, then Developer Settings Click the Personal access tokens menu, then click Generate new token.





2. Write a name for your token.

- Select 'Only Select Repositories'.

- Select the repositories that you want to work on with Kodoshin.





3. Give 'Read Only' permission to 'Content'.





4. Give 'Read and Write' permission to webhooks (Webhooks are essential for activating real time synchronization between github code changes and AI processing)





5. Generate the token





6. Copy the token





7. Copy the token in your profile settings
