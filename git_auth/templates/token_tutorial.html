Generate github token



1. Navigate to your Git account settings, then Developer Settings Click the Personal access tokens menu, then click Generate new token.





2. Select repo as the scope. Select admin:repo_hook if you want to activate real time synchronization.

- Tokens (classic) : The token will be applicable for all the specified actions in your repositories.

- Fine-grained tokens : The token will be applicable for all the specified actions only in the repositories that you choose.





3. Click Generate Token. GitHub will display the personal access token only once. Ensure that you copy the token and store it in a safe space.





4. Now you will add the token in the secure fields in your settings form.
